# Basic-C-Knowledge

A collection of fundamental C programming concepts, examples, and notes for learning and reference.

## Contents

- Core syntax and language basics
- Pointers and memory management
- Data structures (arrays, structs, linked lists)
- File I/O
- Common algorithms and patterns

## Getting Started

### Prerequisites

- A C compiler (`gcc` or `clang`)
- `make` (optional, for building examples)

### Compile and Run

```bash
gcc -Wall -o example example.c
./example
```
